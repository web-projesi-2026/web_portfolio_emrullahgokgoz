Hamburger menü üç katmanlı yapıda tasarlandı: (1) Üç <span> elementinden oluşan ikon, CSS transform animasyonuyla açık/kapalı durumlarında ✕ ikonuna dönüşüyor. (2) position: fixed ile tam ekran kaplayan mobil nav overlay. (3) JavaScript ile .open class toggle'ı ve body overflow kontrolü.
