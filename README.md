# VogueLine

VogueLine, premium moda ürünlerini sergilemek için hazırlanmış statik bir e-ticaret vitrin projesidir.

## Özellikler
- Ürün listeleme
- JSON üzerinden ürün verisi okuma
- Arama ve sıralama
- Favorilere ekleme
- Sepet yönetimi
- Responsive tasarım
- LocalStorage kullanımı

## Kullanılan Teknolojiler
- HTML5
- CSS3
- JavaScript (Vanilla JS)
- JSON
- LocalStorage

├── data/

## Yapılan Düzenlemeler
- Sayfa başlıkları kontrol edildi.
- Favicon tanımları doğrulandı.
- Kırık navigasyon bağlantıları düzeltildi.
- README.md dosyası güncellendi.

## Çalıştırma
Projeyi indirip `index.html` dosyasını tarayıcıda açmanız yeterlidir.
